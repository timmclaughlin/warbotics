# Welcome

This site is the Warbotics knowledge base. Every doc in this folder gets rendered
as a page and indexed into Cloudflare AI Search so it is searchable alongside the
WPILib docs and anything else we crawl later.

Sign in with Slack to get a personal search instance that remembers your
questions and — eventually — lets you see what teammates are asking.
